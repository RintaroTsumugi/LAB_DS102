Bài lab này sử dụng dataset Chest X-Ray để thực hiện bài toán phân loại ảnh y tế bằng SVM.

🧠 Dataset: Chest X-Ray
Chest X-Ray Images (Pneumonia) là bộ dữ liệu ảnh X-quang ngực phổ biến dùng trong nhận diện bệnh viêm phổi.
Dataset gồm các ảnh thuộc hai lớp: NORMAL và PNEUMONIA.

Dataset được cung cấp tại: 👉 https://www.kaggle.com/datasets/paultimothymooney/chest-xray-pneumonia
